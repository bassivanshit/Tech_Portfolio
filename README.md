# 🚀 Futuristic Cyberpunk Portfolio — Vanshit Bassi

A high-performance, visually stunning portfolio website designed for a **Data Analyst & ML Engineer** persona. This project features a "Scifi-Data" aesthetic with custom SVG animations, Three.js backgrounds, and a glitch-themed UI.

## 🛠️ Tech Stack & Features

### **Visuals & Design**
- **Cyberpunk UI:** Dark mode primary (`#03060f`) with Cyan and Violet neon accents.
- **Interactive Background:** Powered by **Three.js** for a dynamic particle/grid effect.
- **Custom Animations:** 
    - Glitch text effects for brand identity.
    - Floating skill orbits (SVG-based).
    - Biometric scan-line animation on the hero character.
- **Micro-interactions:** Custom dual-ring cursor and glassmorphism cards.

### **Functional Sections**
- **Skill Matrix:** Categorized tabs for Data Science, ML, and Tools.
- **Dynamic Timeline:** Scroll-reveal experience tracker.
- **Project Showcase:** Hover-reactive cards with tech-stack badges.
- **Responsive Engine:** Fully optimized for Mobile, Tablet, and Desktop using CSS Grid/Flexbox and `clamp()` typography.
